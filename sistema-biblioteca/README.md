# 📚 Sistema de Biblioteca (Python)

Proyecto sencillo en Python que permite
gestionar una biblioteca desde la consola.

## Funcionalidades
- Agregar libros
- Mostrar libros
- Registrar usuarios
- Prestar libros
- Devolver libros
- Eliminar libros

## Tecnologías
- Python


## Cómo ejecutar
python biblioteca.py
